使用教程：

✔️ 主要可以把输入账号密码规划成一个按钮，你就只需要到注册页面点击开始即可，他会自动处理验证码（cf验证等等、防真人模拟点击）

✔️ 他会自动填充邮箱账号密码（会自动匹配格式，适配有些机场不用让你输入后缀，他会自动解决）

✔️ 他会自动解决注册完之后自动填充登录界面的账号密码以及自动点击自动登录

❌ 他目前有点不能适配某些特殊的机场（类似steam那样的图形验证貌似没办法解决）
